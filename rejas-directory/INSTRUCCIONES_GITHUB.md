# Guía Rápida: Cómo actualizar tu web en GitHub

Este documento es una chuleta para recordar cómo subir cambios, publicar imágenes y encontrar tu enlace de tu página.

## 1. Cómo actualizar tu página (añadir fotos nuevas, textos o cambios que hicimos)

Cuando tengamos archivos modificados o fotos nuevas en la carpeta de tu ordenador, haz esto para que aparezcan en internet:

1. Ve a la página de tu repositorio en **[GitHub](https://github.com/)**.
2. Dale al botón pequeño **"Add file"** (Añadir archivo) y luego a **"Upload files"** (Subir archivos).
3. Abre tu carpeta local: `C:\Users\admin\.gemini\antigravity\scratch\rejas-directory`
4. Selecciona **los archivos modificados** (ej. `script.js`, `index.html`) y las **imágenes nuevas**. *Si no recuerdas cuáles son, arrastra todo, GitHub solo actualizará lo que cambió.*
5. Arrástralos hacia la pantalla de GitHub.
6. Dale al botón verde abajo del todo: **"Commit changes"** (Guardar cambios).
7. **Espera 3 minutos** y recarga (F5) la página de tu web pública para ver los cambios.

## 2. Dónde encontrar el enlace público (Tu link en internet)

Si alguna vez pierdes o no sabes cuál es la dirección de tu web:

1. Ve a tu repositorio en GitHub.
2. Haz clic en la pestaña **⚙️ Settings** (Configuración) arriba de todo.
3. En el menú izquierdo, busca la opción **Pages**.
4. Arriba del todo verás un recuadro que dice: *"Your site is live at..."*.
   - Por ejemplo: `https://tu-usuario.github.io/directorio-rejas/`
   - **Ese es el enlace que puedes enviar a la gente.**

## 3. Plan B: Netlify Drop (La alternativa súper rápida)

Si alguna vez GitHub te da fallos y quieres subir la web en 10 segundos para ver cómo queda en internet:

1. Ve a [app.netlify.com/drop](https://app.netlify.com/drop)
2. Arrastra la carpeta **COMPLETA** de tu proyecto (`rejas-directory`).
3. Te generará un enlace aleatorio al instante (ej: `https://palabra-inventada.netlify.app`).
